# PokerFace 🎭

Онлайн-игра «не улыбнись» с веб-камерами (iOS/Android). Улыбнулся → жёлтая карточка,
второй раз → красная и вылет. Побеждает последний с «железным лицом».

См. [ARCHITECTURE.md](ARCHITECTURE.md) — полная техническая архитектура.

## Структура

```
big-game/
├── apps/mobile/        # приложение (React Native + Expo)
├── server/game/        # игровой сервер (Colyseus, авторитарный)
├── packages/shared/    # общие типы и конфиг (клиент + сервер)
└── prototype/          # браузерный прототип детектора улыбки (тюнинг на ПК)
```

## Запуск (тест на ПК, без телефонов)

Нужен Node.js. Установка зависимостей — один раз из корня:

```bash
npm install
```

Дальше в отдельных терминалах:

```bash
npm run server     # игровой сервер :2567 (+ монитор http://localhost:2567/monitor)
npm run app        # приложение в браузере, http://localhost:8081
npm run bots       # боты для авто-матча (BOTS=6 npm run bots — больше)
```

### Как протестировать игру

- **Соло-флоу:** открой `http://localhost:8081`, введи имя → лобби. Открой **вторую вкладку**
  как второго игрока. Оба «Готов», host жмёт «Начать игру».
- **В игре** пока DEV-режим: кнопка «😀 Улыбнуться» вместо камеры (реальный детект —
  Этап 1). Нажал дважды → красная → вылет. Остался один → победа.
- **С ботами:** запусти `npm run bots`, чтобы наполнить комнату.

### Видео (LiveKit) — настройка

1. Создай бесплатный проект на https://cloud.livekit.io
2. Project Settings → Keys → создай API Key.
3. Скопируй `server/game/.env.example` в `server/game/.env` и впиши `LIVEKIT_URL`,
   `LIVEKIT_API_KEY`, `LIVEKIT_API_SECRET`.
4. Перезапусти `npm run server`.

Без ключей игра работает, но в лобби вместо камер — баннер «видео выключено».

### Прототип детектора улыбки

```bash
cd prototype && node server.js   # http://localhost:5173
```
