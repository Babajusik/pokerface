## Где мы сейчас (готово 🟢)
- Ядро игры: лобби → отсчёт → раунд → карточки → вылеты → победитель.
- Детект улыбки на устройстве (MediaPipe, самохостинг модели).
- Видео LiveKit (лобби+игра) — **работает с VPN** (сеть РФ режет LiveKit Cloud).
- Авто-клип «момент провала», шеринг.
- Хостинг 24/7: Render + GitHub (Babajusik/pokerface), автодеплой.
- Архитектура: меню → создать / найти (список+код) / настройки.
- Доработки: отсчёт, звуки, реванш, ИИ-ведущий (банк шуток + голос, уровни off/normal/savage),
  арсенал «провокатора», retention-меню (быстрая игра, стрик, матчи, онлайн, позвать друзей).
- Дизайн v2 «ночное шоу» (лайм).

## Фаза A — Стабилизация и QA 🔴  ← начинаем здесь
Цель: пройти по каждому экрану, починить баги, добавить состояния ошибок и краёв.

### A1. Аудит по экранам (баги/UX)
- [ ] **MainMenu**: пустое имя, очень длинное имя, быстрая игра когда нет лобби, поведение invite на разных браузерах.
- [ ] **CreateGame**: валидация полей, дабл-тап «Создать», возврат «Назад» во время connecting.
- [ ] **LobbyList**: пустой список, неверный код, лобби исчезло во время входа, поллинг не течёт после ухода.
- [ ] **Lobby**: смена хоста при выходе хоста, готовность/анредигиди, старт <2 игроков, код виден/копируется.
- [x] **Game**: аудит пройден — фикс зарядов арсенала (вылетевшая цель + клиентский кулдаун), стоп голоса при выходе; серверная логика (карточки/вылет/победитель/таймер реплик/hostLevel) проверена — чисто.
- [ ] **Settings**: сохранение/применение порога улыбки, звук/голос тумблеры.

### A2. Сетевые края
- [ ] Реконнект при обрыве WS (Colyseus `allowReconnection` + клиент).
- [ ] Вход в полную комнату / закрытую комнату → понятная ошибка.
- [ ] Хост вышел → корректная передача/закрытие.
- [ ] Сервер: валидация всех сообщений, базовый rate-limit.

### A3. Состояния и обратная связь
- [ ] Камера запрещена → понятное сообщение + запасная кнопка.
- [ ] Видео LiveKit недоступно (без VPN) → явный баннер «нужен VPN/скоро свой сервер».
- [ ] Лоадеры/ошибки везде, без «мёртвых» экранов.

**Готово, когда:** прошли все экраны вдвоём, список багов пуст, нет тупиков.
